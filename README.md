# Finite-element-analysis-program-for-planar-triangles-based-on-MATLAB
本项目为上课的小作业，是一个基于杆单元的三角形网格有限元分析程序，可以任意改变荷载作用位置以及网格的密度。部分代码参考了CSDN上面博主猿力觉醒_的项目代码，在此感激不尽。
